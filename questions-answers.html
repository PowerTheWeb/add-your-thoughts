Questions NO: 1. What is computer programming?

Ans: This is a process of doing a certain computation using coding in an attempt to build or design a computer program.

Questions NO: 2. What is coding?

Ans: This is a process of writing code in order to create or design a computer program.

Questions NO: 3. What is the difference between Programming and coding?

Ans: Coding forms part of programming in that the code that is written is understood by the machine. Programming, on the other hand, creates the program that the code wrote.

Questions NO: 4. What is the difference between a designer and a developer?

Ans: A designer focuses on the front-end and the visuals of a program, looking at the usability of the program. A developer focuses on the code that puts together the program.

Questions NO: 5. Which coding language is the easiest to learn?

Ans: HTML, CSS tend to be the easiest coding languanges to learn for beginners.

Questions NO: 6. Which coding language is the quickest to learn?

Ans: Because HTML and CSS are the easiest to learn, they are also the quickest to master.

Questions NO: 7. What is HTML?

Ans: HTML stands for Hypertext Markup Language and it is not generally considered a coding language because it does not make use of varaibles, conditional statements and iterative loops.

Questions NO: 8. What is CSS?

Ans: CSS stands for Cascading Style Sheets and it is a language used to describe the display of an HTML document.

Questions NO: 9. What is SQL?

Ans: SQL stands for Structured Query Language and this language is used when working wtith databases by letting the a user access and manipulate a database.

Questions NO: 10. What is Python?

Ans: This is a high-level programming language and it is used to build websites and software and it can be used to conduct analysis.
