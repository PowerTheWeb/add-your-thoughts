4. What is Open-Source?

It is a type of software that is released under a license in which the copyright holder grants users the rights to use, study, change, and distribute the software and its source code to anyone and for any purpose. Open-source software may be developed in a collaborative public manner.

5. What is Software?

It is a set of computer programs which is usually associated with documentation and data. It works along side but is different from hardware in which the system is built upon and performs its functions.

6. What is Hardware?

It is the physical components of a computer and help perform the function given by the computer. Examples of hardware include the central processing unit (CPU), the random access memory (RAM) and the motherboard.

10. What is a Compiler?

A compiler is a type of software that translates computer code written in one programming language into another language. This terms usually refers to programs that take source code from a high-level programming language and translates it to a lower level language to create an executable program.
